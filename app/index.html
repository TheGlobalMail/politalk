About Poli/talk

Politalk identifies frequently-used phrases in Australian Federal Parliament. It's an ongoing analysis of Hansard transcripts with the aim of making what is said more transparent and (possibly) more entertaining.

The data was sourced and aggregated by OpenAustralia from the digital copies of Hansard provided by the government. Politalk uses an automated text-mining algorithm to extract the phrases. This code is open-source and available on Github. See our post on Partylines to learn more about our methodolgy.

Please note: this tool is for exploring the data only — please be careful drawing conclusions based only on the phrases listed. The text-mining algorithm is completely automated, and occassionaly, it will generate some misleading and sometimes humorous results. Always refer to Partylines or OpenAustralia to fully understand the context that the phrase was used in.
