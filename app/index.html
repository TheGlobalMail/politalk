About Poli/talk

Politalk makes the political more personal – it builds on the success of our PartyLines tool, which lets you explore how political parties use different words to shape key issues, to let you search by individual senators or members of parliament.

See how The Global Mail is using Politalk →

Politalk searches for issues mentioned by the individual speakers and time-frame you select. Please be careful drawing conclusions based only on the phrases listed. The text-mining algorithm is completely automated, and occasionally it will generate some misleading and sometimes humorous results. The tool leads you to a refined search so you can context on OpenAustralia or see trends on PartyLines.

Using Hansard records crunched by OpenAustralia, the tools include every word (of about 80 million, and counting) spoken in Parliament since 2006. The database is updated each morning by loading the previous day’s Hansard.

Politalk uses an automated text-mining algorithm to extract the phrases. This code is open-source and available on Github. See our post on PartyLines to learn more about our methodology.

Occasionally there are issues with importing the government data; where we find them we fix them as soon as we can. Please let us know if you find any bugs yourself.
